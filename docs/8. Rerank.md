### Cohere Rerank

重排器会根据文档与查询的相关性重新排序，`relevance_score` 越高表示相关性越强，通过`top_n`参数可以控制返回结果的数量。

**示例代码**

```python
from langchain_cohere.rerank import CohereRerank

query = "苹果手机"
documents = [
    "苹果是一种常见的水果，富含维生素C。",
    "汽车是现代交通的重要工具，使用汽油或电力驱动。",
    "苹果公司是一家美国科技企业，生产iPhone等电子产品。",
    "水果种植需要适宜的气候条件和土壤环境。",
]

rerank = CohereRerank(
    base_url="http://rerank", model="rerank-v4.0-fast", cohere_api_key="abc"
)
reranked_docs = rerank.rerank(query, documents)

print("原始排序 | 分数 | 内容")
print("-" * 50)
for doc in reranked_docs:
    print(f"{doc['index']:^8} | {doc['relevance_score']:^12.4f} | {doc['document']['page_content']}")
```

**示例代码执行输出的结果**

```text
原始排序 | 分数      | 内容
--------------------------------------------------
   2    | 0.9500    | 苹果公司是一家美国科技企业，生产iPhone等电子产品。
   0    | 0.1200    | 苹果是一种常见的水果，富含维生素C。
   3    | 0.0800    | 水果种植需要适宜的气候条件和土壤环境。
   1    | 0.0500    | 汽车是现代交通的重要工具，使用汽油或电力驱动。
```
