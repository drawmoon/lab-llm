# 🧪 Lab-LLM

> 一个用于探索、集成与实践各种大语言模型（LLM）工具的实验室，包含 LangChain、LangFlow、LangGraph、LangExtract 等框架的使用示例与实战项目。

---

## 🚀 项目简介

**Lab-LLM** 是一个聚合性实验仓库，旨在帮助开发者快速上手并深入理解 LLM 生态中的核心工具。  
这里包含从基础示例到多工具集成的完整实践，涵盖 **Prompt 构建、工作流设计、文本解析、智能代理、数据处理** 等方向。

无论你是在探索 LLM 的应用边界，还是在搭建自己的智能系统，这个实验室都能帮助你 **快速学习、验证与构建**。

---

## 🧩 涵盖的工具

| 分类 | 工具 / 框架 |
|------|--------------|
| **工作流编排** | [LangFlow](https://github.com/logspace-ai/langflow)、[LangGraph](https://github.com/langchain-ai/langgraph) |
| **提示词与链式构建** | [LangChain](https://github.com/langchain-ai/langchain)、[OpenAI](https://platform.openai.com)、[Ollama](https://ollama.ai) |
| **文本提取与解析** | [LangExtract](https://github.com/langextractor/langextract) |
| **向量数据库与嵌入** | Chroma、FAISS、Milvus、Pinecone |
| **智能体与工具集成** | Function Calling、自定义 Tool、Agentic Workflow |

---

## 📂 仓库结构

```bash
Lab-LLM/

```
